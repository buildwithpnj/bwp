# User Journey Map (USER_JOURNEY.md)

This document maps out the user experience flow, visual checkpoints, and conversion milestones of visitors landing on the **BuildWithPNJ** brand.

---

## 1. Landing Experience Timeline

```
  [ 0–5 seconds ] ──────────► [ 5–15 seconds ] ──────────► [ 15–30 seconds ]
  Visual Hook                 Core Capability Check        Detailed Case Studies
  ("Wow.")                    ("Serious AI systems.")       ("Explore more.")

  ┌───────────────────────────────────────────────────────────────┘
  ▼

  Proof & Telemetry           Action / Inquiry
  ("Trust this person.")      ("Contact BuildWithPNJ.")
```

---

## 2. Milestone Metrics

1. **0–5 seconds (First Impression)**: The dark background void and monospace typography create a professional developer environment.
2. **5–15 seconds (Capabilities Check)**: Bullet points clearly outline the platform's core competencies: AI Agents, Voice systems, and System Automations.
3. **15–30 seconds (Discovery Phase)**: The visitor reviews case studies and project grids to evaluate design patterns.
4. **30–60 seconds (Credibility Check)**: Live telemetry metrics showcase active system logs and contributions.
5. **60+ seconds (Conversion Phase)**: The visitor triggers a CTA to contact the team or try out the sandbox.
