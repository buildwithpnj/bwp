# Unified Design System Spec (UNIFIED_DESIGN_SYSTEM.md)

This document outlines the visual principles, design goals, layouts, and typography rules of the **BuildWithPNJ** brand.

---

## 1. Unified Visual Rationale
BuildWithPNJ is ONE product. The authenticated dashboard (Warborn OS) is a direct extension of the public branding website. To prevent visual discrepancies, the interface layout (margins, shadows, borders, grids) must remain cohesive across both environments.

---

## 2. Page Directory Map

### Public Pages (BuildWithPNJ)
- **Home**: Pitching capability grids and telemetry terminals.
- **Projects & Project Details**: Showcasing case studies and live statistics.
- **Labs & Lab Details**: Dynamic R&D category index.
- **Engineering Journal & Details**: Reading layouts with active scroll indicators.
- **Mission Control**: System-wide telemetry monitors.
- **About / Contact / Newsletter**: Core conversions and onboarding channels.

### Dashboard Pages (Warborn OS)
- **Dashboard Home**: Developer operating center.
- **Notes / Books / Habits**: Daily capturing and tracking workspaces.
- **Finance / Storage / Tools**: Systems control panels.
- **AI Workspace / Agent Inbox**: Local LLM orchestrations.

---

## 3. Typography Hierarchy Rules
- **Brand Display Font**: `Pixel Operator Bold`
  - *Usage*: Hero titles, statistics, Mission Control meters, section headings, and branding wordmarks.
- **Body & Copy Font**: `Geist` (Sans-Serif)
  - *Usage*: Descriptive text blocks, cards, UI lists, buttons, sidebar links, forms, and input fields.
