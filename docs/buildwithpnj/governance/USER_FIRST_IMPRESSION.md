# User First Impression Spec (USER_FIRST_IMPRESSION.md)

This document establishes the user experience benchmarks, sensory pathways, and psychological indicators that shape a visitor's initial impression of the **BuildWithPNJ** brand.

---

## 1. The 10-Second Impression Timeline

A visitor's experience during the first 10 seconds is mapped to specific UX outcomes:

- **0–2 seconds (Core Visuals)**:
  - *Visitor Action*: Page loads.
  - *Sensory Experience*: Deep navy background void `#050816` with the large display header `BUILDING AI SYSTEMS THAT OPERATE.` in `Pixel Operator Bold`.
  - *Outcome*: Instantly signals a high-quality, professional technical environment.
- **2–5 seconds (Narrative Clarity)**:
  - *Visitor Action*: Reads subtext.
  - *Sensory Experience*: Clean paragraph text in the `Geist` font.
  - *Outcome*: Visitor understands the core value proposition: We build production-ready AI systems, not template portfolios.
- **5–10 seconds (Interaction Focus)**:
  - *Visitor Action*: Hovers over CTA or scrolls.
  - *Sensory Experience*: Smooth page transitions and button hover states.
  - *Outcome*: Signals system responsiveness and build quality.

---

## 2. Professional Trust Metrics

To build credibility with technical decision-makers:
- **No Fluff**: Keep the copy concise and technical, using clear terminology instead of generic buzzwords.
- **Proof of Work**: Highlight live telemetry stats and active GitHub logs in the Mission Control panel.
- **System Stability**: Ensure the interface is optimized for performance, loading within 1.5 seconds to showcase execution speed.
