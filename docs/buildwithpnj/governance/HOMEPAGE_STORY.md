# Homepage Story Spec (HOMEPAGE_STORY.md)

This document establishes the narrative flow, visual sections, content hierarchy, and storytelling structure of the **BuildWithPNJ** brand homepage.

---

## 1. Landing Narrative Flow

The landing experience is organized into a clean, multi-layered layout that guides visitors through the platform:

```
  ┌──────────────────────────────────────────────────────────┐
  │  SECTION 1: Hero (Cinematic background, bold headline)    │
  └───────────────────────────┬──────────────────────────────┘
                              ▼
  ┌──────────────────────────────────────────────────────────┐
  │  SECTION 2: Capabilities (AI Agents, Voice AI, Autons)   │
  └───────────────────────────┬──────────────────────────────┘
                              ▼
  ┌──────────────────────────────────────────────────────────┐
  │  SECTION 3: Showcase (Featured Projects grids)            │
  └───────────────────────────┬──────────────────────────────┘
                              ▼

  └───────────────────────────┬──────────────────────────────┘

  ┌──────────────────────────────────────────────────────────┐
  │  SECTION 5: Telemetry (Mission Control statistics)       │
  └──────────────────────────────────────────────────────────┘
```

---

## 2. Page Sections breakdown

### Section 1: Hero Segment
- **Visuals**: Fullscreen header with a 24-second looping cinematic dark abstract coding background.
- **Copy**: Large display title in `Pixel Operator Bold`:
  `BUILDING AI SYSTEMS THAT OPERATE.`
- **CTA**: High-contrast action button: `[Enter Workspace / Get Started]`.

### Section 2: What We Build (Capabilities Grid)
- A 3-column layout highlighting key capabilities:
  - **AI Agents**: Self-correcting workflows.
  - **Voice AI**: Low-latency spoken language integrations.
  - **System Automations**: Workflow automations.

### Section 3: Featured Projects
- Grids showcasing active production-ready systems. Each card displays:
  - Project Title and Tagline.
  - Interactive screenshots and tech stack tags.
  - Link to GitHub repository and live demo.

### Section 4: Engineering Journal
- A listing page preview showing deep technical write-ups. Establishes expertise and authority through detailed, structured sharing.

### Section 5: Mission Control (Live Telemetry)
- Active telemetry dashboards displaying live GitHub logs and CPU/memory statistics.

### Section 6: Newsletter & Footers
- Simple email capture form followed by clean, multi-column navigation links.
