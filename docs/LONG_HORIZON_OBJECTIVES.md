# Long-Horizon Goals & Objective Loops

Describes persistent multi-day objectives scheduling checkpoint review rules.

## 1. Goal Lifecycle
- Objectives detail stop conditions and target progress percentages.
- Progress updates dynamically compute verified checkpoint ratios.

## 2. Checkpoint Verification
- Chron scheduler triggers checkpoint reviews.
- Evidence checks verify if conditions are met to log verification.
- Action logs require admin approvals if risk boundaries are high.
