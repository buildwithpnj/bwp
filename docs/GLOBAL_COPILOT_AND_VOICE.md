# Global Copilot & Voice Layer

Describes persistent AI dashboard assistant and audio transcript triggers.

## 1. Context Injection
- Copilot mounts at layout level to persist conversations across route navigations.
- Injects current URL route path, selected records context, and active workflow states to customize replies.

## 2. Voice Input
- Push-to-talk converts audio waveforms using speech-to-text.
- Decodes commands to propose matching actions or navigations.
