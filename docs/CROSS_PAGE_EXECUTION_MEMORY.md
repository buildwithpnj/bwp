# Cross-Page Continuity Operating Threads

Describes persistent workspaces maintaining task coherence across layout navigations.

## 1. Operating Thread Lifecycle
- Users create bounds for tasks (e.g. "Investigating repeated workflow failure").
- Integrates multiple target logs (workflow runs, objectives history) onto the thread summary payload.

## 2. Pause & Resume Workspace
- Supports archiving active continuity threads to restore workspace focuses later without repetitive context explanation.
