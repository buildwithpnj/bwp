# Agent Presence Intelligence & Suggestions

Describes route intent classifiers regulating copilot visibility.

## 1. Route Intent Classification
- Classifies routes to identify user context status:
  - stuck: Offers recovery details (e.g. diagnostics route).
  - executing: Remains silent (e.g. active workspace route).
  - reviewing: Emits goals reports (e.g. objectives route).

## 2. Suggestion Chips
- Proposes page-sensitive actions dynamically (e.g. "Explain last error", "Verify current checkpoint").
