## Problem
Showcasing detailed macro-photography of luxury jewellery items requires layout systems that dynamically adapt to multiple viewport widths while maintaining the exact asset aspect-ratios.

## Solution
We designed a lightweight catalog website:
- **Pure CSS Grids**: Avoided heavy component library templates, opting for a clean, vanilla stylesheet.
- **Micro-Interactions**: Subtle scale transitions on hover and fade transitions on card layouts to make the luxury products pop.
